# Numpad Emulation (Emulate Numpad)

Temporarily enables the “Numpad Emulation” feature in Blender.

## Features
- Hold or toggle mode
- Customizable hotkey
- Lightweight and simple

## Installation
1. Download the .py file
2. Open Blender → Preferences → Add-ons
3. Click “Install from disk…” and select the file
4. Enable the add-on

## Usage
Press the assigned key to enable "Emulate Numpad".

## Notes
Useful for laptops and keyboards without a numeric keypad.
Default key (~) overrides Blender's View Pie Menu.
You can change the hotkey in addon settings.

The add-on was developed using the Gemini CLI agent
